# Constitutional Modules

Constitutions are behavioral and cognitive constraints for agents.

They are different from skills:

- Skills provide domain knowledge.
- Harnesses provide task execution gates.
- Constitutions constrain reasoning behavior.
- L0 state preserves persistent architectural intent.

Load `constitutions/core.yaml` for all substantial tasks. Add `sui-safety.yaml` for Move/PTB work and `anti-drift.yaml` for long-horizon refactors.
